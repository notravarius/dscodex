FAQ


  • What is DSCodex?

    It's an open source interactive application to manipulate dataframes, get insights from data, create models and whatever you want, without writing any line of code. Just write a request on plain-english and get it done!

  • How it works?

    When you type a request, that request works as input in a natural language processing model that returns a python script. Then, the app executes that script and shows the output on the screen. That model was developed by OpenAI and its name is Codex. Currently, it's on beta so to use it you will need to have your own private keys. Join the codex waitlist here

  • How to use it?

    First you will need to upload a dataset in csv format. Then, you will be able to type a request. As output, you will get: a preview of the original dataset, output for your request, how the dataframe looks after running your request and the generated python script. You will be able to copy the script and save your new dataset!
